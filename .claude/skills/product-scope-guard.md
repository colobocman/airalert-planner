# Product Scope Guard

Use this skill whenever adding a feature.

Ask:
1. Does this improve the planning assistant MVP?
2. Does it strengthen the time-series analytical core?
3. Can it be tested quickly?
4. Does it risk delaying the core deliverable?

Default decision:
- Core analytics/tests/CLI: yes.
- Telegram: only after core is complete.
- Web app/realtime/geospatial engine/neural model: no for MVP.
